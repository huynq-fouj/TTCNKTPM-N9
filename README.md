## Getting Started
 - Clone the repository:
 ```bash
> git clone https://github.com/huynq-fouj/TTCNKTPM-N9
```
## Frontend

- Dowload Node and run:

```cmd
> cd frontend
> npm install
> npm run dev
```
## Backend

- Dowload Maven and run:

```cmd
> cd backend
> mvn clean install
> mvn spring-boot:run
```
